ULTRAMAN AUTONOMOUS SEO

An autonomous, data-driven, production-grade SEO optimization workflow for AI Coding Agents. Loop audits, branch changes, and DevTools validation automatically.

git clone https://github.com/your-username/ultraman-autonomous-seo.git

Autonomous Core Capabilities

1. Data-Driven Auditing

Direct integration with Google Search Console MCP. Retrieves real CTRs, query impressions, sitemap indexes, and identifies low-performing keywords automatically.

2. Safe Git Branching

Never commits directly to main. Automatically branches to seo/gsc-audit-YYYY-MM-DD, applies optimized fixes, runs typescript compilations, and verifies builds before merge.

3. Render Verification

Uses Playwright/Chrome DevTools MCP to spin up local servers and verify the final rendered DOM—ensuring React/Vue/Next.js dynamic tags match expectations.

4. Loop Orchestration

Supports scheduled automation loops via GitHub Actions or local crontabs. The AI wakes up, audits, commits, merges, and triggers deployment automatically.

Interactive Autonomous Architecture

Click on any step below to expand detailed execution parameters, CLI commands, and check criteria.

1
Trigger Source

Automation Schedule

GitHub Actions cron scheduler, local cron scheduler (crontab), or manual developer request wakes up the AI Agent in the workspace.

Execution Mode:
- Cron triggers CLI: npx -y @google/antigravity-cli@latest run --instruction "..."
- The agent is supplied with the ultraman-autonomous-seo plugin rules.
2
Audit Phase

Fetch GSC Performance

The agent reads .seo-config.json, verifies search-console MCP tools, and runs queries to fetch performance data.

MCP Tool Calls:
- get_performance_summary(site_url, period="28d")
- get_search_analytics(site_url, dimensions=["query"], row_limit=100)
- list_sitemaps(site_url)
3
Analysis Phase

Google Guideline Analysis

Agent analyzes data against Google rules: categorizes queries into Critical (CTR < 2%), High Opportunity, and Growth tiers.

Audit Checklist:
- Check title tag lengths (30-60 chars).
- Check meta descriptions (120-160 chars).
- Verify JSON-LD schemas (SoftwareApp, FAQPage, Articles).
- Check hreflang alternates (for N languages, verify N+1 alternates).
4
Modification

Git Branch & Coding

Creates a safe, date-stamped branch seo/gsc-audit-YYYY-MM-DD. Generates/updates JSON-LD schemas, titles, descriptions, and translation files.

Git Sequence:
git checkout main && git pull
git checkout -b seo/gsc-audit-2026-06-18
Changes: Edits page components, updates locale files, and runs sitemap script.
5
Verification

Production Build Verification

Executes the configured build command to capture any TypeScript compilation errors or broken imports before committing.

Build Command:
- Run npm run build (or configured alternative).
- Check exit code. Must be 0.
6
Verification

DevTools DOM Verification

Starts local dev server, calls playwright / chrome-devtools MCP to navigate to localhost, extracts live rendered DOM, and validates tags.

DOM Extraction Script: Runs JS evaluation to check for document.title, canonical links, exact number of hreflang tags, and parses application/ld+json blocks to verify structured data correctness.
7
Deployment

Merge, Push & Deploy

Merges the verified branch back into main, pushes to origin, which triggers the target CI/CD (Vercel, Cloudflare, Netlify) to deploy the SEO fixes.

Deployment Commands:
git checkout main
git merge --no-ff seo/gsc-audit-2026-06-18
git push origin main
8
Re-indexing

Request Google Re-indexing

Submits changed URLs to Google Search Console to request accelerated crawling, so new SEO improvements are indexed within 48 hours.

API calls: Sends indexing requests to GSC for the homepage and key updated subpages.

Installation & Setup

Cross-platform Installer (Node.js - Windows/Mac/Linux) 
# Clone and run install.js
git clone https://github.com/your-username/ultraman-autonomous-seo.git
cd ultraman-autonomous-seo
node install.js

Create a .seo-config.json file in the root of your website repository to specify parameters for the AI Agent:

Project Root: .seo-config.json 
{
  "site_url": "https://your-domain.com/",
  "repo_path": "/Users/username/Code/your-website-repo",
  "build_command": "npm run build",
  "dev_command": "npm run dev",
  "dev_url": "http://localhost:5173",
  "supported_languages": ["en", "zh", "ja"],
  "key_pages": [
    { "path": "/", "schema_types": ["WebSite", "Organization"], "priority": "homepage" },
    { "path": "/faq", "schema_types": ["FAQPage"], "priority": "high" }
  ]
}

Configure your AI Agent to run on a weekly schedule.

GitHub Actions Workflow (.github/workflows/auto-seo.yml) 
name: Ultraman Auto SEO Loop

on:
  schedule:
    - cron: '0 0 * * 0' # Every Sunday at midnight
  workflow_dispatch:

jobs:
  seo:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0 # Need full history to merge branches

      - name: Set up Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Install dependencies & Build
        run: npm ci

      - name: Set up GCP Credentials
        run: |
          echo "${{ secrets.GCP_SEO_KEY_JSON }}" > gcp-credentials.json
          echo "GOOGLE_APPLICATION_CREDENTIALS=$GITHUB_WORKSPACE/gcp-credentials.json" >> $GITHUB_ENV

      - name: Run AI Agent
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
          CLAUDE_API_KEY: ${{ secrets.CLAUDE_API_KEY }}
        run: |
          # Command to invoke your AI Agent CLI
          npx -y @google/antigravity-cli@latest run \
            --instruction "Audit website SEO using GSC, implement fixes in a branch, verify build, and merge to main."
